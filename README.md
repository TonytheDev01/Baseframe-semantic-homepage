# Baseframe-semantic-homepage
BASEFRAME is a clean, semantic HTML5 starter structure built to reinforce best practices in layout, accessibility, and document organization.
This project serves as a reusable foundation for future web projects and experiments.

PROJECT PURPOSE
The goal of BASEFRAME is to:
Practice writing semantic HTML
Establish a solid, reusable page structure
Improve understanding of accessibility-friendly layouts
Create a reliable starting point for CSS and JavaScript projects
